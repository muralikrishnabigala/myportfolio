Unsupervised Machine Learning for Managing Safety Accidents in Railway Stations.

The project titled "Unsupervised Machine Learning for Managing Safety Accidents in Railway Stations" focuses on analyzing accident data using machine learning to enhance safety measures. Data is collected from railway departments and local police stations, including information such as accident type, location, time, environmental conditions, and possible causes like system failures or human negligence. The data is preprocessed and analyzed using unsupervised learning techniques like K-Means and DBSCAN clustering to identify hidden patterns and high-risk zones. Dimensionality reduction methods like PCA and t-SNE are used to visualize trends, while anomaly detection algorithms such as Isolation Forest and One-Class SVM help highlight rare and critical incidents. These insights assist in identifying root causes—whether due to technical issues or operational errors—and guide actionable recommendations such as improved maintenance, operator training, or infrastructure upgrades. The project supports a proactive, data-driven approach to accident prevention, aiming to improve safety protocols across railway stations.
